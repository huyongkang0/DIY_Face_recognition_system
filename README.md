# DIY_Face_recognition_system
《数字图像处理》课程设计，基于深度学习的人脸识别项目。采用三层卷积池化+全连接神经网络。
本项目为三分类人脸识别，含人脸框及识别结果展示。
本项目参考了http://tumumu.cn/2017/05/02/deep-learning-face/
